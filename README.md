# InputManager
 Java Processing, Input Manager listen system
 
 This is a super basic implmentation of an event listener in Java, not meant for production but a quick showcase of how to make a basic listen system
 
 Utilizes basic Game Update pattern, Singleton patter, Event Listener pattern, with simple physics
 
 This code is not meant to be organized, or used in any way. This was practice, done in 20 minutes.
